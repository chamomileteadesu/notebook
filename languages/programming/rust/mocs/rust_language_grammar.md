# Rust Language Grammar

## Data types

- Scalar
  - Integer
    - i8
    - i16
    - i32
    - i64
    - i128
    - isize
    - u8
    - u16
    - u32
    - u64
    - u128
    - usuze
  - Floating-point number
    - f32
    - f64
  - Boolean
    - bool
  - Character
    - char
- Compound types
  - array(`[T; N]`)
  - tuple(`(T, U, ...)`)
  - slice(`&[T]`)
- String
- &str
- Vector(`Vec<T>`)
- Option(`Option<T>`)
- Result(`Result<T, E>`)
- Other
  - Unit(`()`)
  - Never(`!`)

## Exception types

- std::error::Error
  - ParseIntError
  - ParseIntError
  - TryFromIntError
  - ParseFloatError
