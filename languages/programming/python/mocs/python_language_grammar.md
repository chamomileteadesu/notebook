# Python Language Grammar

## Data types

- Numeric
  - int
  - float
  - complex
  - bool(extends int)
- Sequence
  - str
  - list
  - tuple
  - range
- Binary
  - bytes
  - bytearray
  - memoryview
- Set
  - set
  - frozenset
- Mapping
  - dict
- Other
  - object
  - None
  - Ellipsis(...)
  - NotImplemented

## Exception types

- Exception
  - SyntaxError
  - RuntimeError
  - TypeError
  - ValueError
  - ZeroDivisionError
  - OverflowError
  - IndexError
  - KeyError
  - AttributeError
  - StopIteration
  - FileNotFoundError
  - FileNotFoundError
  - IsADirectoryError
  - NotADirectoryError
  - OSError
  - ModuleNotFoundError
  - ImportError
  - NotImplementedError
  - NameError
  - UnboundLocalError
  - CancelledError
